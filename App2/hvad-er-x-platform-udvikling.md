# Hvad er X-platform udvikling

## 1. Definition
X-platform udvikling er en metode, hvor en fælles kodebase
bruges til at udvikle applikationer, som kan køre på flere platforme
fx iOS, Android, Windows og web.

## 2. Hvilke fordele og ulemper er der ved X-platform udvikling?
### Fordele
- En kodebase til flere platforme
- Hurtigere udviklingsproces
- Ensartet funktionalitet på tværs af platforme

### Ulemper
- Lavere performance end native apps
- Sværere at udnytte platformsspecifikke funktioner
- Begrænset adgang til hardware og sensor
- Flere abstraktionslag mellem software og hardware

## 3. Hvornår er native apps bedst egnet?
### Native udvikling er bedst egnet til
- Apps med høje sikkerhedskrav (fx MitID og bankapps)
- Apps der er afhængige af hardware og sensorer
- Companion apps som er et supplement til et andet system (wearables eller IoT-enheder)
