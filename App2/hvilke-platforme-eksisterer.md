# Mobile platforme

## 1. Hvilke programmeringssprog bruges på platformen?
### iOS
- Primært Swift, sekundært Objective-C
- Understøtter også C og C++ via systembiblioteker

### Android
- Primært Kotlin, sekundært Java
- Understøtter C, C++ og Rust via Android NDK

## 2. Hvilke IDE’er og værktøjer bruges til udvikling?
### iOS
- Xcode (Interface Builder, SwiftUI, Simulator, Instruments)

### Android
- Android Studio (Gradle, Emulator, Profiler, Layout Editor)

## 3. Hvordan afvikles applikationer?
### iOS
- Swift og Objective-C kompileres Ahead-Of-Time til native maskinkode
- Apps kører direkte på iOS uden virtuel maskine

### Android
- Kotlin/Java kompileres til bytecode
- Koden afvikles i ART (Android Runtime)
- Systemnære dele i C/C++/Rust kompileres til native kode

## 4. Styrker og svagheder
### iOS
**Styrker**
- Ensartet hardware og OS-versioner

**Svagheder**
- Lukket økosystem
- Kræver macOS til udvikling
- Begrænset systemadgang

### Android
**Styrker**
- Mere fleksibel og åben platform
- Stort hardware-udvalg
- Dybere systemadgang mulig

**Svagheder**
- Fragmentering af OS-versioner og enheder
- Mere kompleks test og vedligeholdelse

## 5. Udbredelse og brug
- Android har den største globale markedsandel
- iOS er særligt udbredt i Europa, USA og blandt premium-brugere
